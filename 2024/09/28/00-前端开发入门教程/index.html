HTML

HTML 超文本标记语言 —— HyperText Markup Language

  • 超文本：链接
  • 标记：标签，带尖括号的文本

标签

标签结构

  • 标签要成对出现，中间包裹内容
  • <> 里面放英文字母（标签名）
  • 结束标签比开始标签多 /
  • 标签分类：双标签 和 单标签

eg：

标签的关系

  • 作用：明确标签的书写位置，让代码格式更整齐

  1. 父子关系（嵌套关系）：子级标签换行且缩进（Tab键）

  2. 兄弟关系（并列关系）：兄弟标签换行要对齐

HTML 基本骨架

  • html：整个网页
  • head：网页头部，用来存放给 浏览器 看的信息，例如：CSS
  • body：网页主体，用来存放给 用户 看的信息，例如：图片、文字
  • title：网页标题

eg：

  • 快速生成的办法：! (英文) + Enter / Tab 键

在 VS Code 中，添加 / 删除 注释的快捷键：Ctrl + /

标题标签

一般用在 新闻标题、文章标题、网页区域名称、产品名称 等等

eg：

标签名：h1 ~ h6（双标签）

显示特点：

  • 文字加粗
  • 字号逐渐减小
  • 独占一行（换行）

技巧：

  1. h1 标签在一个网页中只能用一次，用来 放新闻标题或网页的 logo
  2. h2 ~ h6 没有使用次数的限制

选择器

作用：查找标签，设置样式

基础选择器

标签选择器

  • 标签选择器：使用 标签名 作为选择器 → 选中 同名标签设置相同的样式
  • 特点：选中同名的标签设置相同的样式，无法差异化同名标签的样式

eg：

类选择器

  • 作用：查找标签，差异化 设置标签的显示效果

  • 步骤：

    • 定义类选择器 → .类名
    • 使用类选择器 → 标签添加 class="类名"

eg：

  • 注意：

    1. 类名 自定义，不要用纯数字或中文，尽量用英文命名
    2. 一个类选择器 可以供多个标签使用
    3. 一个标签可以使用多个类名，类名之间用 空格 隔开

id 选择器

  • 作用：查找标签，差异化 设置标签的显示效果

  • 场景：id 选择器一般 配合 JavaScript 使用，很少用来设置 CSS 样式

  • 步骤：

    • 定义 id 选择器 → #id名
    • 使用 id 选择器 → 标签添加 id= "id名"

  • 规则：同一个 id 选择器在一个页面只能使用一次

eg：

通配符选择器

  • 作用：查找页面 所有 标签，设置相同样式
  • 通配符选择器： *，不需要调用，浏览器 自动 查找页面 所有 标签，设置相同的样式

eg：

  • 通配符选择器可以用于 清除标签的默认样式，例如：标签默认的外边距、内边距

文字控制属性

字体大小

  • 属性名：font-size
  • 属性值：文字尺寸，PC 端网页最常用的单位 px
  • 注意：谷歌浏览器默认字号是 16px

字体样式

是否加粗

  • 属性名：font-weight
  • 属性值：
    • 正常：400 / normal
    • 加粗：700/ bold

是否倾斜

  • 作用：清除文字默认的倾斜效果
  • 属性名：font-style
  • 属性值
    • 正常：normal
    • 倾斜：italic

eg：

行高

  • 作用：设置多行文本之间的距离

  • 属性名：line-height

  • 属性值

    • 数字 + px
    • 数字（当前标签font-size属性值的倍数）

  • 行高的测量方法：从一行文字的最顶端（最底端）量到下一行文字的最顶端（最底端）

eg：

行高 - 垂直居中

  • 垂直居中技巧：行高属性值等于盒子高度属性值

eg：

  • 注意：该技巧适用于单行文字垂直居中效果

字体

  • 属性名：font-family
  • 属性值：字体名

eg：

  • font-family 属性值可以书写多个字体名，各个字体名用 , 隔开，执行顺序是 从左向右 依次查找
  • font-family 属性最后设置一个字体族名，网页开发建议使用 无衬线字体 sans-serif

eg：

font复合属性

  • 使用场景：设置网页文字公共样式，简化代码
  • 复合属性：属性的简写方式，一个属性对应多个值的写法，各个属性值之间用 空格 隔开
  • font：是否倾斜 是否加粗 字号/行高 字体（必须按顺序书写）

eg：

  • 注意：字号 和 字体值 必须书写，否则 font 属性不生效

文本缩进

  • 作用：首行缩进，段落空两格
  • 属性名：text-indent
  • 属性值：
    • 数字 + px
    • 数字 + em（推荐：1em = 当前标签的字号大小）

eg：

文本对齐方式

  • 作用：控制内容水平对齐方式

  • 属性名：text-align

  • 属性值：

    属性值 效果
    left 左对齐（默认）
    center 居中对齐
    right 右对齐

eg：

  • text-align 本质是 控制内容的对齐方式，属性要设置给内容的父级

文本修饰线

  • 属性名： text-decoration

  • 属性值：

    属性值 效果
    none 无
    underline 下划线
    line-through 删除线
    overline 上划线

eg：
