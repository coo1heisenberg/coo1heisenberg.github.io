网络数据传输的基本原理

  • ip地址
    • 唯一确定目的主机
  • port 端口号
    • 唯一确定进程
  • 协议
    • UDP协议：无连接的不可靠的协议
    • TCP协议：面向连接的可靠的协议

  1. 创建发送端的socket对象
  2. 把要发送的数据封装成数据报包
  3. send方法发送数据报包
  4. 释放资源close

接收端步骤

  1. 创建接收端的socket对象
  2. 创建用于接收的数据报包
  3. receive方法接收数据
  4. 解析数据报包
  5. 释放资源close
